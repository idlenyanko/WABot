A WhatsApp bot built with Node.js, designed for automation and fun commands!  

## Features
- Custom commands (`/commands`)
- Data storage & management (`/data`)
- Event listeners (`/listeners`)
- Helpers for utilities (`/helpers`)

## Setup
1. Install Node.js & Git
2. Clone this repo:  
   ```bash
   git clone https://github.com/idlenyanko/WABot.git
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Run bot:  
   ```bash
   node index.js
   ```
