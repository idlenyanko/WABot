## Setup
1. Install Node.js & Git
2. Clone this repo:  
   ```bash
   git clone https://github.com/idlenyanko/WABot.git
   ```
3. Install dependencies:  
   ```bash
   npm install
   ```
4. Run the bot:  
   ```bash
   node index.js
   ```
